### Boolean Algebra Properties

**Commutative Properties**
A+B = B+A
A* B = B* A

**Associative Properties**
A + (B +C) = (A+B) + C
A * (B * C) = (A * B) * C

**Distributive Properties**
A + (B * C) = (A + B) * (A + C)
A * (B+C) = A * B + A * C

**De Morgen's Theorem**
(A+B)' =A' * B' and (A * B)' = A' + B' 

### Boolean Identities

**Multiplicative Identity**
A * 1 = A   
A * 0 = 0
A * A = A
A * A' = 0 

**Additive Identity**
A + 0 = A
A + 1 = 1
A + A = A
A + A' = 1
